Comandos para Repositorio remoto

  • git remote add *alias_repo* *url* # agregar un repositorio remoto
  • git remote -v # ver los repositorios remotos
  • git remote set-url *alias_repo* *url* # cambiar la url de un repositorio remoto
  • git push -u *repo-remoto* *nombreRama* # subir cambios a un repositorio remoto
  • git push # subir cambios a un repositorio remoto
  • git pull # traer cambios de un repositorio remoto
  • git pull *repo-remoto* *rama* # traer cambios de un repositorio remoto
  • git clone *url* # clonar un repositorio remoto
